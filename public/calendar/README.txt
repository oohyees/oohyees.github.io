CS 保研 DDL 日历订阅
====================

订阅地址
--------
https://oohyees.github.io/calendar/csbaoyan-ddl.ics

Google Calendar 使用方式
------------------------
1. 打开 Google Calendar
2. 左侧边栏找到「其他日历」→ 点击「+」→「通过网址添加」
3. 粘贴上述 ICS URL
4. 点击「添加日历」

Apple Calendar 使用方式
-----------------------
1. 打开「日历」应用
2. 菜单栏 →「文件」→「新建日历订阅」
3. 粘贴上述 ICS URL
4. 点击「订阅」，可调整自动刷新频率（建议每天）

Outlook 使用方式
----------------
1. 打开 Outlook 日历
2. 「添加日历」→「从 Internet 订阅」
3. 粘贴上述 ICS URL
