Place licensed Century Gothic webfont files here:

```text
public/fonts/century-gothic/centurygothic.woff2
public/fonts/century-gothic/centurygothic.ttf
public/fonts/century-gothic/centurygothic_bold.woff2
public/fonts/century-gothic/centurygothic_bold.ttf
```

The site registers them as `Century Gothic` in `src/styles/theme.css`.

Do not commit a system or Microsoft Office font file unless its license allows
web distribution from this public GitHub Pages repository.
