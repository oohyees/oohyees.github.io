## 2. LaTeX 数学公式

### 行内公式

爱因斯坦质能方程 $E = mc^2$ 可能是最著名的物理公式。欧拉公式 $e^{i\pi} + 1 = 0$ 则被誉为最优美的数学公式。

### 块级公式

傅里叶变换：

$$
\hat{f}(\xi) = \int_{-\infty}^{\infty} f(x) e^{-2\pi i \xi x} dx
$$

矩阵乘法：

$$
\begin{bmatrix}
a & b \\
c & d
\end{bmatrix}
\begin{bmatrix}
x \\
y
\end{bmatrix}
=
\begin{bmatrix}
ax + by \\
cx + dy
\end{bmatrix}
$$

Bayes 定理：

$$
P(A|B) = \frac{P(B|A) \cdot P(A)}{P(B)}
$$

### 多行公式

$$
\begin{aligned}
\nabla \times \vec{\mathbf{B}} - \frac{1}{c} \frac{\partial\vec{\mathbf{E}}}{\partial t} &= \frac{4\pi}{c}\vec{\mathbf{j}} \\
\nabla \cdot \vec{\mathbf{E}} &= 4\pi\rho \\
\nabla \times \vec{\mathbf{E}} + \frac{1}{c} \frac{\partial\vec{\mathbf{B}}}{\partial t} &= \vec{\mathbf{0}} \\
\nabla \cdot \vec{\mathbf{B}} &= 0
\end{aligned}
$$

## 3. 代码块 + 数学结合

算法复杂度分析中常用数学记号：

```ts diff
function fibonacci(n: number): number {
- // O(2^n) — exponential
+ // O(n) with memoization
- return n <= 1 ? n : fibonacci(n - 1) + fibonacci(n - 2);
+ const memo = new Map<number, number>();
+ function fib(k: number): number {
+   if (k <= 1) return k;
+   if (memo.has(k)) return memo.get(k)!;
+   const result = fib(k - 1) + fib(k - 2);
+   memo.set(k, result);
+   return result;
+ }
+ return fib(n);
}
```

时间复杂度从 $O(2^n)$ 降到 $O(n)$，空间复杂度 $O(n)$。

## 总结

这就是 MDX 相比普通 Markdown 多出的核心能力：

- **嵌入 Astro 组件**：响应式表格、自定义 UI 等，普通 `.md` 做不到
- **LaTeX 数学公式**：`remark-math` + `rehype-katex` 渲染，支持行内和块级公式
- **代码块增强**：Shiki 的 diff、highlight 语法在 `.mdx` 中同样可用
